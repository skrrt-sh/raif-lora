# RAIF LoRA — CUDA / unsloth stack for the RTX 5070 Ti (Blackwell, sm_120).
#
# Blackwell is the fragile part: older torch builds emit no sm_120 kernels and
# fail at runtime with "no kernel image is available". You need CUDA 12.8-class
# wheels. The reliable install order is:
#
#   1) python -m venv .venv-cuda && source .venv-cuda/bin/activate   (or: uv venv)
#   2) pip install -r cuda/requirements.txt
#   3) Verify Blackwell is seen:
#        python -c "import torch; print(torch.__version__, torch.cuda.is_available(), torch.cuda.get_device_capability())"
#      Expect a torch >=2.7, True, and capability (12, 0). If capability prints
#      (12, 0) but is_available() is False, your driver predates Blackwell —
#      update the NVIDIA driver, not these packages.
#
# Versions below are FLOORS chosen for Blackwell support as of 2026-01; prefer
# the latest of each. If `pip install unsloth` pulls a torch without cu128
# kernels, reinstall torch explicitly from the cu128 index (the --extra-index
# line handles that here).
